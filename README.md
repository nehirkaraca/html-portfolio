# Personal Portfolio Website
This is a simple personal portfolio website created to showcase my skills, projects, and background as a computer engineering student.

## About the Project
The website includes several sections such as:
- Introduction / Banner
- Skills
- Portfolio / Projects
- Contact information

The goal of this project was to practice building a structured, clean website using HTML and CSS, while learning layout, styling, and responsive design basics.

## Technologies Used
- HTML5
- CSS3

## Features
- Clean and simple layout
- Background images and styled sections
- Responsive design basics

## What I Learned
- Structuring pages with HTML
- Styling layouts with CSS
- Working with background images and positioning
- Creating reusable sections and lists
- Basic web design principles
